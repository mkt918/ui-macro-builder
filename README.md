# ⚡ UI Macro Builder

Excel VBA を **ブロックを組み立てるだけ** で作れる、授業用のビジュアルプログラミング学習ツール。
ループ・条件分岐・配列のアルゴリズムを、目で見ながら学べます。

🔗 **使う**: https://mkt918.github.io/ui-macro-builder/

---

## 生徒のみなさんへ — 使い方

1. **🎮 クエスト選択** ボタンから、挑戦するクエストを選ぶ
2. プリントに **流れ図（フローチャート）** を書く
3. 左のブロックを中央のエリアに **ドラッグ＆ドロップ** して組み立てる
4. 右の **仮想 Excel** で、ブロックがどう動くか確認する
   - **▶ 実行**: 全部を自動で動かす（速度はスライダーで調整）
   - **⏭ ステップ**: 1つずつ進めて、何が起きるか確かめる
   - **↺ リセット**: 最初に戻す
5. うまくいったら **🎉 クエストクリア！** が出る
6. 詰まったら **💡 ヒント**（3段階）を順番に見る
7. どうしても分からなければ **📖 答えを見る**（確認のあと表示）
8. できた VBA コードは **📋 コピー** して、本物の Excel に貼り付けて実行！

### ブロックの種類
| カテゴリ | できること |
|---------|-----------|
| 🟦 セル操作 | セルに値を入れる・取り出す・コピー・消す |
| 🟧 繰り返し | 同じことを何回もくり返す（ループ） |
| 🟪 条件分岐 | もし〜なら、で処理を分ける |
| 🟩 書式設定 | 背景色を変える・太字にする |
| 🟢 配列 | データを「はこの列」にためる |
| 🟥 シート操作 | シートを追加・切り替え |
| 🔵 値 | 数字・テキスト・計算 |

🌙/☀️ ボタンで **ダークモード / ライトモード** を切り替えられます。

### クエストの追加
Claude に以下のように頼むだけで `src/tasks/tasks.js` に追記されます。
```
こういうクエストを追加して：
- タイトル: 〇〇
- 難易度: 1〜5
- 目標: 生徒に何をさせたいか
- 答え: （イメージがあれば。なければ自動生成）
```
ヒント3段階・クリア自動判定も一緒に作られます。
